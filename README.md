# stat_vector_search

## overview
- 統計調査の目的をベクトル検索したらよさげかもしれない
- 統計調査の目的は事前に調査計画のPDFからテキストを抽出して正規表現でひっこぬいてます
  - `purpose.csv` がそうです。

## Getting Started
### 1. docker-compose up !!
``` shell
git clone 
cd stat_vector_search
docker-compose up -d
```

>❗⏰ take a few minutes for First Start Only   
> - model download
> - purpose csv to vector


### 2. browser Access
http://localhost:7979/docs/
